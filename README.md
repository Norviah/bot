## Bot

A template project for a Discord bot using Discord.js, TypeScript, and the [Akairo](https://discord-akairo.github.io/#/) framework. A guide documenting the structure of this project and how to extend it and/or custom make commands/modules will come *eventually*,
to currently run as it is right now, you **must** edit the [config](./config.example.ts) file to have the correct values and tokens. Once the correct values are set, rename the file to `config.ts`, then:
  - build the project with `npm run build`,
  - and run it with `npm run start`
